Replacable vs Non-Replacable Elements

Replacable Elements

These are HTML elements whose content is replaced by an external resource. Their appearance and intrinsic dimensions (like width and height) are determined by the resource they load, not by the CSS formatting model of the document itself.

  • Image (img)
  • Video (video)
  • Audio (audio)
  • Object (object)
  • Embed (embed)
  • IFrame (iframe)
Non-Replacable Elements

These are HTML elements whose content is contained directly within the document itself, and their rendering is entirely determined by the CSS formatting model and the text content/child elements they enclose.

  • Paragraph (p)
  • Heading (h1, h2, h3, etc.)
  • Span (span)
  • Div (div)
  • List (ul, ol, dl)

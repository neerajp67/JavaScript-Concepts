CSS Specificity

CSS specificity determines which styles are applied to an element when there are conflicting styles.

It is calculated based on the types of selectors used in a rule:

  • Inline styles have the highest specificity.
  • ID selectors have higher specificity than class selectors.
  • Class selectors have higher specificity than type selectors.

For example, if an element has both a class and an ID selector applied, the styles from the ID selector will take precedence.

If two selectors have the same specificity, the one that appears last in the CSS file will be applied.

To see the effects of specificity, you can modify the styles in the < code>style.css file by commenting or uncommenting specific rules.

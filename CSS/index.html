CSS Practice

Types of CSS

  • Inline CSS (style applied directly to elements)
  • Internal CSS (style defined within the <style> tag in head)
  • External CSS (style defined in an external file)

Type of Selectors

  • Universal Selector
  • Type (Element) Selector
  • Class Selector
  • Id Selector
  • Attribute Selector
  • Descendant Selector
  • Child Selector
  • Adjacent Selector
  • General Sibling Selector
  • Pseudo-classes
  • Pseudo-elements
  • Group Selector
  • Another Group Selector
  • Negation Pseudo-class
  • Structural Pseudo-classes
  • Dynamic Pseudo-classes
